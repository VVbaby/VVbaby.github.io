二、图片隐写分类

1.不需要工具
右击属性 文件末尾藏有字符串 文件包含 修改文件头
（只要一个jpg文件出现FFD8 开头出现FFD9 结尾就不会影响读取存储）

2.通过图种
GIF png/bmp JPG

3.不论什么图片 需要一些脚本或特征明显
双图隐写 stegpy silenteye

4.破坏文件头会识别错误 破坏文件尾不一定

5.在010 editor上修改宽会显示错误 一般修改长

三、细讲
1.strings查找可打印的字符 kali预装 strings file
grep使用正则表达式搜索，并输出匹配行 grep flag
file识别文件类型 file文件

2.文件包含
binwalk分离文件
kali预装 binwalk file binwalk -e file windows
此处可查询https://www.cnblogs.com/pcat/p/5256288.html

foremost提取文件
foremost file
此处也可使用dd 例如

打开kali 输入dd if-binwalk,formost,dd.png of-out.jpg bs-1 skip-271007
其中bs-1为数量

此处可查询https://www.cnblogs.com/yuanqiangfei/p/9138625.html

foremost -t需要恢复文件类型后缀（如jpg） -i扫描的分区 -o指定存放文件
指定存放文件的文件夹需为空，不然会报错

3.gif
特殊帧 可使用PS/stegsovle 也可看帧的时间间隔

4.png（bmp）
zilb(010/pngcheck)
IDHR、IDAT

lsb隐写/lsb加密（zsteg/stegsolve)
打开stegesolve中data extracct 点击preview可看到隐藏的格式
save iaxt只可整行复制 不如sabe bin可分块复制
具体stegsolve用法可参考
https://www.cnblogs.com/cat47/p/11483478.html

另 打开Kali 输入 zsteg 文件名 可显示出隐藏格式（zsteg -a 文件名会显示更多）
输入 zsteg -e 复制上一步显示出的格式 文件名 >保留新的文件名
